## 关于本项目

SillyGirl 是一个基于 Go 语言的开源机器人框架，以 JavaScript 插件系统为核心，支持多平台适配器与丰富的交互能力。开发者可以通过简单的 JS 脚本快速构建具有定时任务、HTTP 接口、持久化存储等复杂功能的机器人应用。

本项目已不再活跃维护，但代码和文档保持完整开源，供社区参考和使用。
